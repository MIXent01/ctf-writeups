    Weave

    Category: Crypto | Difficulty: Medium

    Challenge Overview

    Description: "every piece of the cipher lies open on the table. compose them correctly and the flag is yours." Core Concept: Rank-Metric Cryptography, McEliece Cryptosystem, Gabidulin Codes, Welch-Berlekamp Algorithm.

    In this challenge, we are presented with a variant of the McEliece cryptosystem utilizing Rank-Metric codes (specifically, Gabidulin codes). However, true to the challenge description, the private key components are explicitly leaked in the provided output.json. Our task is to use these pieces to decode a codeword corrupted by an intentional error (the "frays") and recover the AES key to decrypt the flag.

    Initial Analysis

    Looking at chall.sage, we can identify the environment parameters: * $Q = 2$, $M = 43 \implies$ The finite field is $GF(2^{43})$. * $N = 40 \implies$ The length of the code. * $K = 8 \implies$ The dimension of the code (the secret is 8 elements long). * $\text{FRAYS} = 5 \implies$ The rank of the injected error.

    The encryption process constructs a standard McEliece public key ($\text{warp}$): $$\text{warp} = \text{knot} \cdot \text{loom} \cdot \text{shuttle}^{-1}$$ * $\text{loom}$ (Generator): A Gabidulin Moore matrix defined by $N$ independent $\text{pegs}$. * $\text{knot}$ ($S$): A random invertible $K \times K$ scrambling matrix. * $\text{shuttle}$ ($P$): An invertible $N \times N$ matrix.

    The cipher vector ($\text{bolt}$) is generated by multiplying the $\text{secret}$ by the $\text{warp}$ and adding an error term ($\text{frays}$): $$\text{bolt} = \text{secret} \cdot \text{warp} + \text{frays}$$

    Finally, the secret is hashed using SHA-256 to derive an AES-GCM key, which encrypts the flag.

    The Mathematical Vulnerability

    In a standard McEliece setup, $\text{knot}$, $\text{loom}$, and $\text{shuttle}$ are kept secret. An attacker faces the NP-hard problem of decoding a random linear code.

    However, because the output.json leaks all these components, we can easily strip away the obfuscation. If we multiply both sides of the encryption equation by $\text{shuttle}$ on the right, we get: $$\text{bolt} \cdot \text{shuttle} = (\text{secret} \cdot \text{knot}) \cdot \text{loom} + (\text{frays} \cdot \text{shuttle})$$

    Let's define our substituted variables: 1. $\text{bolt}' = \text{bolt} \cdot \text{shuttle}$ (Our target vector) 2. $\text{secret}' = \text{secret} \cdot \text{knot}$ (Our target message) 3. $\text{error}' = \text{frays} \cdot \text{shuttle}$

    The equation simplifies to a standard decoding problem: $$\text{bolt}' = \text{secret}' \cdot \text{loom} + \text{error}'$$

    Why not just pick random columns?

    In standard Hamming-metric codes, errors affect specific columns. You can usually pick $K$ random columns, hope they are error-free, and invert the matrix to find the secret.

    However, Gabidulin codes use the Rank-Metric. The error here is generated as $u \cdot B$, meaning it spans a 5-dimensional subspace. This "smears" the error across almost all coordinates in the field. The probability of picking 8 random columns that evaluate to zero error is astronomically low. We need an algebraic decoding approach.

    The Exploit: Welch-Berlekamp for Gabidulin Codes

    The Gabidulin code with $N=40$ and $K=8$ has a minimum rank distance of $d = 40 - 8 + 1 = 33$. It can correct up to $t = \lfloor(d-1)/2\rfloor = 16$ errors. Since our error has a rank of $5 \le 16$, decoding is guaranteed to succeed.

    We can apply a linear algebraic approach similar to the Welch-Berlekamp algorithm for Reed-Solomon codes, but adapted for linearized polynomials.

    We want to find an Error Locator polynomial $V(x)$ of degree $\le 16$ and an Evaluator polynomial $N(x)$ of degree $< 16 + 8 = 24$ such that: $$V(\text{bolt}'_j) = N(\text{pegs}_j) \quad \text{for } j = 0, \dots, N-1$$

    By treating the coefficients of $V(x)$ and $N(x)$ as unknowns, we construct an $N \times (16 + 23)$ matrix and find its right kernel (nullspace).

    Once we have $V(x)$ and $N(x)$, we recursively extract the coefficients of the underlying message polynomial $P(x)$, which gives us $\text{secret}'$. Finally, we multiply by $\text{knot}^{-1}$ to recover the original secret and decrypt the AES vault.

    The Solve Script

    Here is the complete SageMath script to decode the Gabidulin code and decrypt the flag:

    import json
from sage.all import *
from Crypto.Cipher import AES
from hashlib import sha256

print("[*] Loading data...")
with open('output.json', 'r') as f:
    data = json.load(f)

spec = data['spec']
Q = spec['q']
M = spec['m']
N = spec['n']
K = spec['k']

print("[*] Setting up GF(2^43)...")
R = PolynomialRing(GF(Q), 'x')
modulus = R(spec['modulus'])
Fqm = GF(Q**M, name='a', modulus=modulus)

def to_bits(e):
    if int(e) == 0: return [0]
    return [int(x) for x in bin(int(e))[2:][::-1]]

def unpack_vec(v):
    return vector(Fqm, [Fqm(to_bits(e)) for e in v])

def unpack_mat(m):
    return Matrix(Fqm, [[Fqm(to_bits(e)) for e in row] for row in m])

def pack(v):
    out = []
    for e in v:
        cs = list(e.polynomial()) if e else []
        cs = [int(c) for c in cs] + [0] * (M - len(cs))
        val = 0
        for i, c in enumerate(cs):
            val |= c << i
        out.append(int(val))
    return out

print("[*] Unpacking matrices...")
warp = unpack_mat(data['warp'])
bolt = unpack_vec(data['bolt'])
pegs = unpack_vec(data['loom']['pegs'])
knot = unpack_mat(data['loom']['knot'])
shuttle = unpack_mat(data['loom']['shuttle'])

iv = bytes.fromhex(data['vault']['iv'])
body = bytes.fromhex(data['vault']['body'])
tag = bytes.fromhex(data['vault']['tag'])

print("[*] Formulating decoding system...")
bolt_prime = bolt * shuttle

# Constructing the Welch-Berlekamp linear system
rows = []
for j in range(N):
    row = []
    # V(x) coefficients
    for i in range(16):
        row.append(bolt_prime[j]**(Q**i))
    # N(x) coefficients
    for k in range(23):
        row.append(-(pegs[j]**(Q**k)))
    rows.append(row)

M_mat = Matrix(Fqm, rows)

print("[*] Computing kernel to find Error Locator & Evaluator...")
ker = M_mat.right_kernel()
sol = ker.basis()[0]

v_i = sol[0:16]
n_i = sol[16:39]

# Find lowest non-zero degree in V
i_min = 0
while i_min < 16 and v_i[i_min] == 0:
    i_min += 1

print("[*] Extracting secret recursively...")
v_i_ext = list(v_i) + [Fqm(0)] * K
P = []
for k in range(K):
    rhs = n_i[i_min + k]
    for step in range(1, k + 1):
        i = i_min + step
        j = k - step
        rhs -= v_i_ext[i] * (P[j] ** (Q**i))

    P_k_q_imin = rhs / v_i[i_min]
    P_k = P_k_q_imin ** (Q**(M - i_min))
    P.append(P_k)

secret_prime = vector(Fqm, P)

print("[*] Reversing obfuscation matrices...")
potential_secret = secret_prime * knot.inverse()

print("[*] Decrypting Vault...")
secret_bytes = b''.join(int(v).to_bytes((M + 7) // 8, 'big') for v in pack(potential_secret))
wrap_key = sha256(secret_bytes).digest()[:16]

cipher = AES.new(wrap_key, AES.MODE_GCM, nonce=iv)
flag = cipher.decrypt_and_verify(body, tag)
print(f"\n[+] Flag successfully recovered: {flag.decode()}")
